개인정보처리방침

시행일: 2026년 5월 11일

StockAI(이하 “회사”)는 개인정보 보호법 등 관련 법령을 준수하며, 서비스 제공에 필요한 최소한의 개인정보를 처리합니다. 본 방침은 회사가 어떤 정보를 수집하고, 어떻게 이용·보관·파기하는지 설명합니다.

1. 개인정보의 처리 목적

회사는 다음 목적을 위해 개인정보를 처리합니다.

  • API 키 발급 및 계정 식별
  • 무료·유료 플랜 이용 한도 관리
  • 결제, 구독 상태 확인, 환불 및 고객지원
  • 서비스 장애 대응, 보안, 부정 이용 방지
  • 법령상 의무 이행 및 분쟁 대응

2. 처리하는 개인정보 항목

구분처리 항목수집 방법
계정/API 키이메일 주소, API 키, 플랜 등급, 가입일시이용자 입력, 서비스 자동 생성
결제/구독결제 이메일, 구독 ID, 구독 상태, 고객 포털 URL, 결제·환불 처리 결과결제대행사 또는 구독 플랫폼 웹훅
이용 기록접속 IP, 요청 경로, 요청 일시, API 이용량, 조회 종목 코드, 오류 로그서비스 이용 과정에서 자동 생성
AI 브리핑 입력종목 코드, 시장 데이터, 이용자가 입력한 포트폴리오 항목이용자 입력 및 외부 데이터 조회
문의이메일 주소, 문의 내용, 처리 내역이메일 또는 문의 채널

회사는 주민등록번호, 건강정보, 정치적 견해 등 민감정보를 의도적으로 수집하지 않습니다. 이용자는 서비스 입력란에 불필요한 개인정보 또는 민감정보를 입력하지 않아야 합니다.

3. 개인정보의 보유 및 이용 기간

항목보유 기간
계정/API 키 정보서비스 이용 기간 동안 보관하며, 계정 삭제 또는 API 키 폐기 요청 시 지체 없이 삭제 또는 비활성화합니다.
구독·결제 기록거래 관계 확인 및 관련 법령상 보존 의무 이행을 위해 필요한 기간 동안 보관합니다.
이용량 및 보안 로그부정 이용 방지와 장애 대응을 위해 최대 1년간 보관할 수 있습니다.
문의 기록문의 처리 완료 후 3년간 보관할 수 있습니다.

4. 개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 경우에는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 따라 수사기관, 법원, 감독기관 등이 적법한 절차로 요구하는 경우
  • 결제·구독 처리를 위해 결제대행사 또는 구독 플랫폼에 필요한 정보가 전달되는 경우

5. 개인정보 처리의 위탁 및 외부 서비스 이용

수탁자 또는 외부 서비스이용 목적처리 정보
Railway 등 클라우드 인프라서비스 서버 및 데이터베이스 운영계정 정보, 이용 기록, 로그
Lemon Squeezy, Toss Payments 등 결제 서비스결제, 구독, 환불, 고객 포털 제공이메일, 결제·구독 정보
Anthropic APIAI 브리핑 생성종목 코드, 시장 데이터, 이용자가 입력한 브리핑 요청 내용

회사는 AI 브리핑 생성 시 이메일, API 키 등 계정 식별정보를 의도적으로 AI 모델에 전송하지 않습니다. 다만 이용자가 직접 입력한 내용에 개인정보가 포함된 경우 해당 정보가 처리될 수 있으므로 주의해야 합니다.

6. 국외 이전 가능성

회사가 이용하는 클라우드, 결제, AI API 서비스는 국외 서버를 사용할 수 있습니다. 이 경우 이전되는 정보는 서비스 제공에 필요한 범위로 제한되며, 회사는 적절한 보호 조치가 있는 서비스를 이용하기 위해 노력합니다.

7. 개인정보의 파기

회사는 개인정보 보유 기간이 경과하거나 처리 목적이 달성된 경우 지체 없이 해당 정보를 파기합니다. 전자적 파일은 복구하기 어렵도록 삭제하고, 출력물은 분쇄 또는 이에 준하는 방법으로 파기합니다. 법령상 보존 의무가 있는 경우 해당 기간 동안 분리 보관할 수 있습니다.

8. 이용자의 권리와 행사 방법

이용자는 언제든지 본인의 개인정보 열람, 정정, 삭제, 처리정지를 요청할 수 있습니다. 요청은 아래 문의처로 이메일을 보내 행사할 수 있으며, 회사는 관련 법령에 따라 지체 없이 처리합니다. 다만 법령상 보존 의무가 있거나 서비스 운영·분쟁 대응에 필요한 경우 일부 요청이 제한될 수 있습니다.

9. 자동화된 의사결정 및 AI 처리

서비스의 AI 브리핑은 입력된 종목 데이터와 시장 지표를 바탕으로 자동 생성됩니다. 이 결과는 이용자의 법적 권리 또는 의무를 결정하지 않으며, 투자 판단을 대신하지 않습니다. 이용자는 AI 브리핑을 참고 자료로만 사용해야 합니다.

10. 쿠키 및 로컬 저장소

서비스는 데모 이용 횟수 표시 등 편의 기능을 위해 브라우저의 로컬 저장소를 사용할 수 있습니다. 이용자는 브라우저 설정을 통해 로컬 저장소와 쿠키를 삭제할 수 있습니다. 서버 측 이용 한도는 별도로 관리됩니다.

11. 안전성 확보 조치

회사는 개인정보 보호를 위해 API 키 기반 인증, 접근 권한 제한, HTTPS 통신, 이용량 기록, 로그 점검 등 기술적·관리적 보호 조치를 시행합니다. 다만 이용자는 API 키를 제3자에게 공유하거나 공개 저장소에 노출하지 않도록 주의해야 합니다.

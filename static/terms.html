StockAI

이용약관

시행일: 2026년 5월 11일

중요 고지: StockAI는 투자자문업, 투자일임업, 금융투자상품 매매추천 서비스가 아닙니다. 서비스가 제공하는 내용은 주식 가격·거래량·기술적 지표 등을 바탕으로 한 정보 브리핑이며, 특정 종목의 매수·매도·보유를 권유하지 않습니다. 모든 투자 판단과 그 결과는 이용자 본인의 책임입니다.

제1조 목적

본 약관은 StockAI(이하 “회사”)가 제공하는 한국 주식 변동 브리핑, API, 구독형 서비스(이하 “서비스”)의 이용 조건, 절차, 회사와 이용자의 권리·의무 및 책임사항을 정합니다.

제2조 용어의 정의

  • “서비스”란 회사가 웹사이트 및 API를 통해 제공하는 주식 데이터 조회, AI 브리핑, 계정·구독 관리 기능을 말합니다.
  • “이용자”란 본 약관에 따라 서비스를 이용하는 개인 또는 법인을 말합니다.
  • “API 키”란 이용자 식별과 이용량 관리를 위해 발급되는 고유 문자열을 말합니다.
  • “유료 플랜”이란 회사가 정한 요금을 결제하고 확장된 이용 한도를 제공받는 구독 상품을 말합니다.

제3조 약관의 게시와 변경

회사는 본 약관을 서비스 화면에 게시합니다. 회사는 관련 법령을 위반하지 않는 범위에서 약관을 변경할 수 있으며, 중요한 변경이 있는 경우 적용일 7일 전부터 공지합니다. 이용자에게 불리하거나 중대한 변경은 가능한 범위에서 30일 전부터 공지합니다.

제4조 계정 및 API 키

  • 서비스는 현재 이메일 기반 API 키 발급 방식으로 운영됩니다.
  • 이용자는 정확한 이메일 주소를 제공해야 하며, 동일 이메일로 결제한 경우 해당 이메일에 연결된 API 키의 플랜이 갱신됩니다.
  • API 키는 이용자 본인이 안전하게 보관해야 하며, 키 유출 또는 제3자 사용으로 발생한 문제는 이용자 책임입니다.
  • 회사는 보안, 남용 방지, 법령 준수를 위해 API 키를 정지하거나 재발급할 수 있습니다.

제5조 서비스의 내용

회사는 주식 가격 데이터, 기술적 지표, 시장 지수, AI 기반 브리핑, 포트폴리오 복기, API 기능을 제공합니다. 서비스의 구체적인 내용과 제공 범위는 플랜, 운영 환경, 외부 데이터 제공 상황에 따라 달라질 수 있습니다.

제6조 투자정보 관련 고지

  • 서비스의 브리핑은 정보 제공 목적이며 투자자문 또는 매매추천이 아닙니다.
  • 회사는 제공 정보의 정확성·완전성·최신성을 높이기 위해 노력하지만, 외부 데이터 오류, 지연, AI 생성 오류가 발생할 수 있습니다.
  • 이용자는 투자 결정 전 공시, 재무제표, 거래소 정보, 증권사 자료 등 원천 자료를 직접 확인해야 합니다.
  • 서비스 이용 결과 발생한 투자 손실, 기회비용, 세금, 수수료 등에 대해 회사는 고의 또는 중대한 과실이 없는 한 책임을 지지 않습니다.

제7조 이용 제한

이용자는 다음 행위를 해서는 안 됩니다.

  • API 키 공유, 우회 접속, 비정상 자동화 등으로 이용 한도를 회피하는 행위
  • 서비스를 투자자문, 리딩방, 불법 유사투자자문, 사기성 광고에 사용하는 행위
  • 서비스 또는 데이터를 무단 복제·재판매·대량 수집하는 행위
  • 허위 정보 입력, 시스템 공격, 보안 취약점 악용, 서비스 운영 방해 행위
  • 관련 법령 또는 공서양속에 반하는 행위

제8조 요금, 결제 및 구독

  • 유료 플랜의 가격, 이용 한도, 기능은 서비스 화면에 표시된 내용을 따릅니다.
  • 결제는 회사가 연동한 결제대행사 또는 구독 결제 플랫폼을 통해 처리됩니다.
  • 유료 구독은 별도 해지 전까지 정기 갱신될 수 있으며, 이용자는 결제 플랫폼의 구독 관리 페이지 또는 회사가 안내하는 방법으로 해지할 수 있습니다.
  • 결제 완료 후 같은 이메일의 API 키가 유료 플랜으로 갱신됩니다.

제9조 청약철회 및 환불

디지털 콘텐츠 및 온라인 서비스의 특성상, 이용자가 유료 기능 또는 API 사용을 개시한 경우 관련 법령상 청약철회가 제한될 수 있습니다. 다만 서비스 장애, 중복 결제, 표시·광고와 다른 제공 등 회사 책임 있는 사유가 있는 경우 회사는 관련 법령과 결제 플랫폼 정책에 따라 환불 또는 보상 절차를 진행합니다.

제10조 서비스 변경, 중단

회사는 운영상·기술상 필요, 외부 데이터 제공 중단, 보안 이슈, 천재지변 등 사유가 있는 경우 서비스의 전부 또는 일부를 변경·중단할 수 있습니다. 회사는 가능한 경우 사전에 공지합니다.

제11조 지식재산권

서비스 화면, 소프트웨어, 문구, API 구조, AI 브리핑 포맷 등 회사가 제공하는 콘텐츠의 권리는 회사 또는 정당한 권리자에게 있습니다. 이용자는 서비스 이용 목적 범위에서만 이를 사용할 수 있습니다.

제12조 개인정보 보호

회사는 서비스 제공을 위해 필요한 최소한의 개인정보를 처리하며, 자세한 내용은 개인정보처리방침을 따릅니다.

제13조 면책

회사는 무료 서비스, 베타 기능, 외부 데이터 오류, AI 생성 결과의 해석 차이, 이용자의 투자 판단으로 발생한 손해에 대해 고의 또는 중대한 과실이 없는 한 책임을 지지 않습니다. 법령상 책임을 배제할 수 없는 경우에는 해당 법령이 정한 범위 내에서 책임을 부담합니다.

제14조 분쟁 해결

본 약관은 대한민국 법령을 따릅니다. 서비스 이용과 관련한 분쟁은 회사와 이용자가 성실히 협의하여 해결하며, 협의가 어려운 경우 관련 법령상 관할 법원 또는 분쟁조정기관의 절차를 따릅니다.
